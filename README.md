<h1 align="center">🍽️ Zomato Data Insights</h1>

<p align="center">
A Python-based Exploratory Data Analysis (EDA) of Zomato's restaurant dataset.
This project focuses on data cleaning, handling missing values, and visualizing food trends.
Key highlights include rating distributions, cost analysis, and geographic insights using Seaborn and Matplotlib to help drive data-backed business decisions.
</p>

---

## 📌 Project Overview

This project analyzes Zomato restaurant data to uncover meaningful business insights through data visualization and statistical exploration.

### Objectives
- Data Cleaning & Preprocessing
- Missing Value Handling
- Restaurant Rating Analysis
- Cost Distribution Analysis
- Online Order & Table Booking Insights
- Location-wise Restaurant Trends
- Customer Preference Analysis

## 📊 Exploratory Data Analysis
⭐ Rating Distribution

Understanding how restaurants are distributed across different rating categories.

💰 Cost Analysis

Analyzing restaurant pricing trends and customer affordability patterns.

🛒 Online Order Analysis

Examining the impact of online ordering on restaurant popularity.

📍 Location-wise Analysis

Identifying restaurant concentration and performance across locations.

## 📈 Key Insights
Most restaurants fall within mid-range ratings.
Popular locations have higher restaurant density.
Online ordering is offered by a significant portion of restaurants.
Cost distribution shows a concentration in affordable and mid-range segments.
Customer preferences vary significantly across locations.
